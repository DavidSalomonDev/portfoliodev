# David Salomón Martínez Valladares — Portfolio

Sitio personal de David Salomón Martínez Valladares (davidsalomon.dev), construido con [Next.js](https://nextjs.org/) y [Chakra UI](https://chakra-ui.com/). Presenta su biografía, experiencia como Cloud Engineer / Data Engineer / Software Developer, portafolio de proyectos y enlaces a sus redes y otros sitios (blog, david-salomon.com).

Desplegado en [Vercel](https://vercel.com/) a partir de este repositorio.

## Desarrollo

```bash
npm install
npm run dev
```

El sitio queda disponible en `http://localhost:3000`.

## Scripts

- `npm run dev` — levanta el servidor de desarrollo
- `npm run build` — genera el build de producción
- `npm run lint` — ejecuta ESLint
- `npm run prettier` — formatea el código con Prettier
